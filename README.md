# Raspberry-WEBCamera
Raspberry based WEB camera with Flask

Prerequisites
1. next packages should be installed on system level with sudo apt install:
 python3.11-dev,
 libcamera-dev,
 libcamera-apps,
 python3-libcamera,
 python3-flask,
 python3-picamera2,
 libjpeg-dev,
 libjpeg62-turbo-dev,
 python3-opencv
